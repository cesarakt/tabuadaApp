## Ferrramentas Utilizadas

* [React Native](https://reactnative.dev/)
* [Yarn](https://yarnpkg.com/)
* [Expo](https://docs.expo.io/)

## Como Utilizar

#### Clone este repositório
```bash
git clone https://github.com/cesarakt/tabuadaApp.git
```

#### Instale as dependências
```bash
yarn install
```

#### Execute a aplicação
```bash
yarn start
```
